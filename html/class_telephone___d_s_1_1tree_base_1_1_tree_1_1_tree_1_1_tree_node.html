Returns
a list of pointers to current node and each recursive child nodes of current node
Order:
take current node as a tree-root-node , and then place this tree in this way:
  1. root-up
  2. among all the child nodes of a node , the child node with least index in the child-nodes list placed in the far left
in this case , the order of the nodes in the returned list is: up-to-down and left-to-right

Parameters
valuethe specified data
equalfunction to judge whether one data is equal to another , which will change the data objects to be compared
Returns
a list of indexes of child nodes containing the specified data
How to compare:
call a function using the callable object passed in by the parameter to compare whether the data part of each child node is equal to the specified data:
  • if equal , the function should return 1
  • if not equal , the function should return 0
  • the function should have two formal parameters , the first one is a lvalue reference bound to type U which means one data object , the second one is a lvalue reference bound to type U which means another data object
Order:
the returned list is consistent with the child-nodes list in sequential order of nodes

template<typename U >
virtual void Telephone_DS::treeBase::Tree::Tree< T >::TreeNode< U >::traverseAsTree ( std::function< void(TreeNode< U > &, int, int)>  doSomething)
inlinevirtual

take current node as a tree-root-node , traverse the tree by rows(include the root-node)

Parameters
doSomethinga function to call when this function traverses each node
Order:
take current node as a tree-root-node , and then place this tree in this way:
  1. root-up
  2. among all the child nodes of a node , the child node with least index in the child-nodes list placed in the far left
in that case , this function will traverse all nodes of the tree row by row from up to down while traversing from left to right in a row
Action:
As the function traverses each node , each time it will call the callable object passed in by the parameter. For the callable object passed in by the parameter , there are the following requirements:
  1. return void
  2. the first formal parameter is an lvalue reference bound to a TreeNode<U> , which means the node being traversed currently
  3. the second formal parameter is a int , which means the index of the row being traversed currently(place the tree as described above , assume that the index of the top row is 0 , and row-index increases downward)
  4. the third formal parameter is a int , which means the index of the node in the current traversed row(place the tree as described above , number each node in the current traversed row , assume that the node-index of the most-left node is 0 , and node-index increases rightward)
  • the formal parameter list can be like this: (TreeNode<U> &nowNode , int rowIndex , int nodeIndex)
