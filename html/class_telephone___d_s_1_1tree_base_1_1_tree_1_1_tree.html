inlinevirtual

traverse current tree and then find the nodes of current tree containing specified data(include the root-node)

Parameters
datathe specified data
equalfunction to judge whether one data is equal to another , which will change the data objects to be compared
Returns
a list of pointers to nodes of current tree containing the specified data
How to compare:
call a function using the callable object passed in by the parameter to compare whether the data part of each node is equal to the specified data:
  • if equal , the function should return 1
  • if not equal , the function should return 0
  • the function should have two formal parameters , the first one is a lvalue reference bound to type T which means one data object , the second one is a lvalue reference bound to type T which means another data object
Order:
place current tree in this way:
  1. root-up
  2. among all the child nodes of a node , the child node with least index in the child-nodes list placed in the far left
in this case , the order of the nodes in the returned list is: up-to-down and left-to-right
